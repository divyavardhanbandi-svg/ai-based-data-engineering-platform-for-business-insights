# AI-Based Data Engineering Platform for Business Insights

## Project Description
This is a simple AI-based business insights project. It loads business data, cleans the data, creates business KPIs, visualizes business performance, and predicts profit using a machine learning model.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## How to Run

### Step 1: Install requirements
```bash
pip install -r requirements.txt
```

### Step 2: Run the project
```bash
streamlit run app.py
```

## Features
- Business data loading
- Data cleaning
- KPI dashboard
- Category-wise sales insights
- Region-wise profit insights
- Region-wise sales insights
- Profit prediction using machine learning
